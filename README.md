# Needlework.Net

![App preview](app-preview.png)

Needlework.Net is an open-source helper tool for the LCU that provides documented endpoints and can send requests without any code setup. Created using .NET! üå†

# Features

- Interactive OpenAPI documentation
- REST data transfer console
- WebSocket event data viewer

# Requirements

- Windows x64
- .NET 8 runtime. [It can be downloaded here.](https://dotnet.microsoft.com/en-us/download/dotnet/thank-you/runtime-desktop-8.0.7-windows-x64-installer)
